1 NLP/LLM 表示与 Tokenization

本模块深入探讨模型如何“看见”并“理解”文本：从原始语料的预处理到离散符号的分词，再到向量空间的表示与位置感知。

1.1 文本规范化与预处理

Unicode 规范化、空白字符处理与安全边界。

1.2 Tokenization（子词/字节级）

BPE、WordPiece、BBPE 等主流分词策略。

1.3 表示学习、Embedding 与位置编码

从 Token 到向量，以及 RoPE 等位置感知的演进。

模块概述

Tokenization 是 LLM 的基石。一个糟糕的分词器会直接限制模型的语言能力和跨语言泛化。本模块将带你从底层理解这些机制，并学会如何选择和评估不同的表示方案。
